# AtlasMemory Examples

AtlasMemory-specific local automation runtime examples retained separately from the reusable template. These files may contain real organization, project, branch, repo, and check names and are not canonical defaults.

Treat these as live-org reference material. Some helper commands and copied configs can mutate labels,
Projects, workflows, or PR state when used from an authenticated runtime. Apply them only inside an installed
runtime, review diffs first, and prefer dry-run/runtime-control commands before enabling `--apply`,
`--publish`, `--merge`, or `--close-issues`.
