# Generic Examples

Placeholder-safe local automation runtime defaults for a new project. Replace `OWNER/REPO`, `OWNER/1`, and command examples before running mutating automation.

Use these files as operator examples, not as a runnable runtime directory:

1. Install or sync `templates/local-automation-runtime/` into your runtime host.
2. Copy only the example snippets you need into that installed runtime.
3. Replace placeholder owners, repos, Project numbers, branch names, checks, and commands.
4. Start with dry runs such as `atlas-agent-plan-queue --dry-run` and `atlas-agent-unattended --cycles 1` before adding mutating flags.
